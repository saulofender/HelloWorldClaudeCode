## Project

Classic Snake game ("Jogo da Cobra") built with vanilla JavaScript and HTML, running entirely in the browser. Requirements are documented in `.llm/prd.md`.

## Running

Open `index.html` directly in a browser — no build step, server, or dependencies required.

## Requirements (from PRD)

- Light/clear color palette
- High-score (maximum points) display
- Speed increases with each level progression
